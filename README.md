# Minecraft_Server - Mod Pack Installation Guide

Welcome! This guide will help you install the Mod pack so you can join our custom Minecraft server. Follow each step carefully to get started.

## Requirements
CurseForge App (recommended) -
Minecraft Java Edition
At least 16 GB Ram 
Internet connection for downloading the modpack

## Installation 

1. Download the CurseForge App
   > https://www.curseforge.com/download/app
   > Install and open the app
2. Go to the Minecraft Selection
   > Select "Minecraft" Under the "Choose Game".
3. Download the zip file from my website or fromn above
   > Https://dylancakalic.com/minecraft_server
   > Download to a location that you will remember
4. Install the mod pack
   > Once inside of "Minecraft" inside of curseforge under "My Modpacks" Choose the import button
   > Then choose import profile .zip
   > Navigate to your zip file and select ok

5. Let Curseforge download the pack for you
   > Note do not update any of the mods in the mod pack
   > The mods have been carefully selected so that they work in harmony with eachother
6. Hit the play button and get ready to play some minecraft!
   > once in minecraft go to multiplayer\n
   > Then select add server
   > under the server address input the code you were given (if you dont have the code reachout on the website)
   > Click Done, then join
